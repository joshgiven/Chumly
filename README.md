## Chumly
Chumly is an activity-centered, platonic match-making service, connecting people based on common interests, locations, and schedules.

### Tech-ables
- Spring MVC
- JPA/JPQL
- MySQL
- JSP/HTML/CSS/Bootstrap
- Amazon Web Services
